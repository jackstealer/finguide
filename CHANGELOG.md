### Features
- **Expense Management**
  - Add/remove expenses manually
  - 8 expense categories (Food, Transport, Entertainment, Shopping, Bills, Healthcare, Education, Other)
  - Load sample data for testing
  - Date-based tracking

- **AI Analysis**
  - Ensemble ML model (85-90% accuracy)
  - Category-wise spending breakdown
  - Daily average calculations
  - Transaction pattern analysis
  - Weekend spending detection
  - Small transaction tracking

- **Smart Recommendations**
  - Category-specific advice
  - Benchmark comparisons
  - Potential savings calculations
  - Pattern-based insights

- **Goal Tracking**
  - Set savings goals with timeframes
  - Achievability analysis
  - Progress tracking
  - Monthly savings calculations

- **Visualizations**
  - Pie chart with non-overlapping labels
  - Bar chart with angled labels
  - Custom tooltips
  - Gradient effects

### Technical
- **Backend**: Flask 3.0, Python 3.11
- **Frontend**: React 18, Recharts 2.10
- **ML**: scikit-learn 1.3.2 (Random Forest + Linear Regression)
- **Data**: pandas, numpy
- **Styling**: CSS3 with animations
